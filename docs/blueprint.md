# **App Name**: Zenit Tracker Basic

## Core Features:

- QA Workflow Interface: Provides an interface for creating test sessions, running test cases, and exporting reports.
- Test Case Importer: Facilitates import of test cases from structured .xlsx files, allowing efficient batch upload of test plans.
- Report Generator: Generates well-formatted Excel reports of test sessions. It contains well-formatted content like results by case.
- Smart Bug Assistant: Utilizes AI tool to identify if test case notes mention related functionality, in order to intelligently determine relevant known issues using available notes, bugId, naReason information.
- Responsive Design: Adapts display and functionality for use across devices, enhancing user experience.
- Session State Persistence: Stores and retrieve test session data using browser storage APIs

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), conveying trust and stability.
- Background color: Light gray (#F9FAFB), providing a clean, modern backdrop.
- Accent color: Electric purple (#A855F7), for highlights and calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text, giving a clear and professional style.
- Use modern, flat icons from Lucide to visually represent different actions and statuses.
- Emphasize a clear, tabular layout for presenting test case data, enhanced with subtle visual cues.
